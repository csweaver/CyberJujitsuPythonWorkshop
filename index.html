Setup

To participate, you'll need to install Anaconda python. This will let you open jupyter notebooks. You will also need to download the lesson file

Windows

Python

  • Download and install Anaconda for Python 2.7.
  • Use all of the defaults for installation.

Python Workshop Jupyter Notebook

Click on the download zip button at the Github Repository. Once this downloads you will need to extract it. Make note of where the extracted files live.

Running

To run your jupyter notebook, Click on the Start Menu > Anaconda > Jupyter Notebook. The jupyter notebook make take a minute or two to start up. When it starts it will open a page in a web browser that says 'Jupyter Notebook' and has a list of all the files and folder on this computer. Click on Downloads > CyberJujitsuPythonWorkshop-master > CyberJujitsuPythonWorkshop-master > CyberJujitsu.ipynb. If you are done early, you can read a little about how to use a jupyter notebook.
